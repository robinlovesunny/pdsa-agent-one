# 智能编码助手通义灵码

## 产品概述
智能编码助手通义灵码（简称为通义灵码），是由阿里云提供的智能编码辅助工具，提供代码智能生成、智能问答、多文件修改、编程智能体等能力，为开发者带来高效、流畅的编码体验，引领 AI 原生研发新范式。同时，为企业客户提供企业标准版、专属版，具备企业级场景自定义、私域知识增强等能力，助力企业研发智能化升级。

## 核心能力
- **代码补全 Code Completion**  
  经过海量优秀开源代码数据训练，可根据当前代码文件及跨文件的上下文，生成行级/函数级代码、单元测试、代码优化建议等。沉浸式编码心流，秒级生成速度，让您更专注在技术设计，高效完成编码工作。
  
- **智能问答 Ask Mode**  
  智能问答模式整合海量研发文档、产品文档、通用研发知识，并结合工程级感知能力，为开发者解决编码过程中遇到的研发问题，协助进行代码问题修复、调试或运行错误排查等。
  
- **文件编辑 Edit Mode**  
  文件编辑模式具备多文件代码修改能力，当开发者需要精准修改代码文件时，能结合需求描述和当前工程环境进行多文件修改，支持多次迭代、代码审查，帮助开发者高效可控地完成代码修改任务。
  
- **智能体 Agent Mode**  
  智能体模式具备自主决策、环境感知、工具使用等能力，可根据开发者的编码诉求，使用工程检索、文件编辑、终端等工具，端到端完成编码任务。支持开发者配置 MCP 工具，使编码更贴合工作流程。

## 产品优势
- **多种会话模式**  
  一次会话流中同时支持问答模式、文件编辑模式、智能体模式，开发者可针对不同场景和问题难度自由切换模式，最大化工作效率。
  
- **工程自动感知**  
  根据任务描述，自动感知工程框架、技术栈、所需代码文件、错误信息等工程内信息，无需手动添加上下文，使任务描述更轻松，代码补全更贴合业务场景。
  
- **工程级变更**  
  可根据任务描述，自主进行任务拆解和工程内多个代码文件修改，支持多次对话逐步迭代或快照回滚，与通义灵码协同完成编码任务。
  
- **记忆感知**  
  支持基于大模型的自主记忆能力，在对话过程中逐步形成针对开发者个人、工程、问题等相关的丰富记忆，越用越懂您。
  
- **多种企业版方案，灵活选择**  
  提供企业标准版、企业专属版等多种方案，支持企业个性化定制，加速企业内智能研发的规模化落地。

## 功能介绍
### 行间代码补全
- **行级/函数级实时续写**  
  根据当前语法和跨文件的代码上下文，自动感知当前工程，实时生成行、函数级代码。
- **注释生成代码**  
  通过注释描述功能需求，直接在编辑器区生成代码，保持编码心流不间断。

### 智能问答
- **研发问题问答**  
  遇到编码疑问或技术难题时，一键唤起通义灵码，无需离开 IDE 客户端，快速获得答案和解决思路。
- **工程问答**  
  通过问答快速结合当前仓库进行工程理解、代码查询，并可通过自然语言描述需求，生成简单需求或缺陷的整体修复建议和代码。
- **图片多模态问答**  
  支持选择、拖拽或粘贴图片作为上下文，自动分析内容，生成代码建议或问题修复建议。
- **企业知识库问答**  
  借助企业知识和数据进行问答，快速构建企业研发知识问答助手，提升团队效率和协作能力。

### 文件编辑
- **工程级变更**  
  根据任务描述，进行工程内多个代码文件修改，支持多次对话逐步迭代或快照回滚，协同完成编码任务。
- **精确编辑**  
  在开发者提供的上下文范围内完成代码文件修改，不会超出预期。
- **快速执行**  
  严格遵循任务描述和上下文进行修改，无需额外复杂任务计划，相比智能体模式更迅速。
- **工具使用**  
  拥有文件读取、工程内语义检索、文件编辑等工具使用能力，帮助快速完成代码修改。

### 编程智能体
- **工程级变更**  
  根据任务描述，自主进行任务拆解和工程内多个代码文件修改，支持多次对话迭代或快照回滚。
- **工程自动感知**  
  自动感知工程框架、技术栈、所需代码文件、错误信息等，无需手动添加上下文。
- **工具使用**  
  可自主使用十多种内置编程工具（如读写文件、代码查询、错误排查），并支持自动感知和使用 MCP 工具。
- **终端命令执行**  
  在执行任务时，自主决策并运行终端命令，大幅提升编码任务执行效率。

### 企业级管理
- **统一授权管理**  
  支持统一为企业内开发者添加使用权限，集中管理成员权限。
- **统计报表**  
  提供开发者活跃度、AI 代码生成占比、问答使用占比等统计数据，帮助企业了解使用效果。
- **知识管理**  
  高效管理企业私域数据，结合数据提供企业知识库问答、企业代码生成增强功能，使编码助手更贴合业务场景。
- **自定义扩展管理**  
  自由组合多种上下文（如代码变更、文件、输入），自定义企业场景常用指令，灵活扩展开发场景。
- **专属 VPC 部署**  
  企业专属版提供专属 VPC 部署，支持企业内网连接，全链路保障数据安全，满足合规要求。
